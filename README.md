## ✨ قابلیت‌های ویژه

| 🚀 **قابلیت** | 📝 **توضیح** |
|---|---|
| **مدیریت کانفیگ** | ساخت، ویرایش، فعال/غیرفعال‌سازی و حذف کانفیگ‌های VLESS با UUID منحصربه‌فرد |
| **سهمیه ترافیک** | تعیین سهمیه ترافیک برای هر کانفیگ (MB/GB) با نمایش لحظه‌ای مصرف |
| **تاریخ انقضا** | تعیین تاریخ انقضا برای هر کانفیگ با هشدار روزهای باقی‌مانده |
| **پروتکل‌های متنوع** | پشتیبانی از VLESS/WS، XHTTP Packet-Up، XHTTP Stream-Up |
| **گروه‌بندی ساب** | ساخت گروه‌های سابسکریپشن با صفحه پابلیک مجزا و قابلیت رمزگذاری |
| **داشبورد حرفه‌ای** | نمایش آمار لحظه‌ای، نمودار ترافیک ساعتی، اتصالات زنده و لاگ‌ها |
| **اتصالات زنده** | نمایش آی‌پی، مدت اتصال و حجم مصرفی هر کاربر به‌صورت لحظه‌ای |
| **QR Code** | تولید QR Code برای هر کانفیگ و گروه سابسکریپشن |
| **تم تاریک/روشن** | تغییر تم با یک کلیک (رنگ‌بندی مشکی-نارنجی) |
| **ذخیره‌سازی خودکار** | ذخیره‌سازی داده‌ها در فایل JSON (اطلاعات پس از ری‌استارت از بین نمی‌رود) |
| **پروکسی HTTP** | پروکسی داخلی برای درخواست‌های HTTP/HTTPS |

## 📊 مزایای TK-Ui نسبت به رقبا

| مزیت | توضیح |
|-------|--------|
| **⚡ سرعت بالا** | به‌دلیل استفاده از FastAPI و عدم وابستگی به دیتابیس سنگین |
| **🎨 طراحی مدرن** | رابط کاربری زیبا با تم مشکی-نارنجی و انیمیشن‌های جذاب |
| **🔒 امنیت بیشتر** | احراز هویت با کوکی امن، هش رمز با SHA-256+SALT |
| **📱 واکنش‌گرا** | پشتیبانی کامل از موبایل و تبلت |
| **🔄 قابلیت توسعه** | کد تمیز، کامنت‌گذاری‌شده و ماژولار |
| **📦 سبک و کم‌حجم** | حجم کم و نیاز کم به منابع سرور |
| **🌍 چندزبانه** | پشتیبانی کامل از زبان فارسی |
